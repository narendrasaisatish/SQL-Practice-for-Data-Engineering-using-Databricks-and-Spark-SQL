# RetailMart Enterprise Data Platform

## Objective

Build a complete Retail Data Engineering project using Databricks.

The project covers:

- SQL (Beginner to Advanced)
- Spark SQL
- Delta Lake
- ETL Pipelines
- Data Warehouse
- Git & GitHub
- Performance Optimization

## Technologies

- Databricks
- Spark SQL
- Delta Lake
- Git
- GitHub

## Business Domain

RetailMart sells products through multiple stores.

Customers purchase products.

Stores maintain inventory.

Employees manage stores.

Suppliers provide products.

Managers use reports to make business decisions.
