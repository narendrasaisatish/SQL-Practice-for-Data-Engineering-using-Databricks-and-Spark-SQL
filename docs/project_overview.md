# RetailMart Data Engineering Project

## Objective

This project demonstrates SQL and Data Engineering concepts using Databricks and Spark SQL.

The project follows a real-world retail business scenario where raw data is ingested, transformed, and analyzed using SQL.

## Technologies

- Databricks
- Spark SQL
- Delta Lake
- Git
- GitHub

## Architecture

Raw Files

Bronze Layer
    ↓
Silver Layer
    ↓
Gold Layer
    ↓
Business Reports
